测试点

  • / 当前页
  • /ip 返回请求IP
  • /uuid 返回 UUID4
  • /user-agent 返回 user-agent
  • /headers 返回 header 字典
  • /get 返回 GET 数据
  • /post 返回 POST 数据
  • /patch 返回 PATCH 数据
  • /put 返回 PUT 数据
  • /delete 返回 DELETE 数据
  • /anything 返回请求数据，包括请求类型
  • /anything/:anything 返回请求数据，包括请求URL
  • /status/:code 返回给定的HTTP状态码
  • /response-headers?key=val 返回给定的响应头
  • /redirect/:n 302 重定向 n 次
  • /redirect-to?url=foo 302 重定向至 foo URL
  • /redirect-to?url=foo&status_code=307 307 重定向至 foo URL
  • /relative-redirect/:n 302 相对重定向 n 次
  • /absolute-redirect/:n 302 绝对重定向 n 次
  • /cookies 返回 Cookie 数据
  • /cookies/set?name=value 设置一个或多个简单的 Cookie
  • /cookies/delete?name 删除由查询字符串提供的 Cookie
  • /stream/:n 流 min(n, 100) 行
  • /delay/:n 延迟响应 min(n, 10) 秒
  • /html 显示一个HTML页面.
  • /robots.txt 返回 robots.txt 规则
  • /deny robots.txt 中禁止访问的页面.
  • /encode_base64/:value base64加密
  • /decode_base64/:value base64解码
